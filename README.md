# Financial Sentiment Analysis using FinBERT

## Overview
This project fine-tunes a transformer-based model to classify financial text into positive, negative, or neutral sentiment.

## Features
- FinBERT-based sentiment classification
- Streamlit web app interface
- Real-time text input analysis

## Tech Stack
- Python
- Hugging Face Transformers
- PyTorch
- Streamlit

## How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
